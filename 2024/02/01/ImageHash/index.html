Image Block Retrieval Using Image Hashing

Project introduction

In this coursework of data structure, We implemented the functionality to locate a given query image block in the source image by developing basic hashing algorithms. Furthermore, we optimized the algorithm’s performance using perceptual hashing, thereby reducing algorithm complexity and improving query efficiency.

Challenge

The course work requires to use perceptual hashing in order to reduce the complexity of the algorithm. This implies the need to convert photos to the frequency domain and perform hash processing in that domain. The task involves hash bucket lookup, DCT transformation, and fast searching, requiring completion of the retrieval task within a specified time. This assesses our understanding and proficiency in algorithm complexity and its practical applications.

Project gains

This marks my initial attempt at tackling a computer vision challenge, presenting a perfect opportunity to enhance my Python programming skills. The project also demands familiarity with DCT, a valuable data processing tool that will contribute to my future research in PPG signal analysis.
